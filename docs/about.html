关于这个项目

ABOUT

智潮 / Signal Tide

这是一个面向 AI 领域的静态日报项目。页面每天整理当日的重要新闻，不直接转贴 RSS 标题，而是重新筛选、重写、编排后发布。

项目定位

它不是新闻流，也不是聚合器首页。它更接近一张日报：头版突出最重要的一条，其余版面按产品、政策、工具、研究和影响分布，尽量给出清楚的层次。

编辑方式

每日先抓取候选源，再人工式筛选主题，最后按统一结构重写到版面里。页面上的标题、导语和正文提要以可读性和准确性为优先，不照搬原标题，也尽量避免空泛表述。

来源范围

主要覆盖官方发布、科技媒体、研究博客、开发者生态和政策治理类信息。每条稿件都会保留来源链接，方便读者回到原始出处继续查证。

发布方式

项目采用纯静态页面，可直接部署在 GitHub Pages。每日更新后会同步生成当期数据、RSS 和归档文件，便于公开访问和后续检索。

多语言结构

UI 文案和文章内容分离维护。界面文本存放在 locale 文件里，文章正文则写入 issues.json，以便同一份日报同时支持多种语言切换。

为什么做归档

日报的价值不只在当天。归档页把每一期保存下来，让这个项目既能作为每日入口，也能作为后续回看某个时间点 AI 行业状态的索引。
